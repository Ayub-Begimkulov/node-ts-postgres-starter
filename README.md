# Usage

```shell
git clone https://github.com/Ayub-Begimkulov/node-ts-postgres-starter.git
cd node-ts-postgres-starter
rm -rf .git
npm install
mv .env.example .env
```

Start for development

```shell
docker-compose -f docker-compose.dev.yml up
```

Start for production

```shell
docker-compose up
```
